## Project Description
The goal of this project is to analyze global sea level rise data since 1880 and predict future sea level change through the year 2050. Using Python, Matplotlib, Pandas, and SciPy, we visualize trends and calculate lines of best fit for both the full dataset and recent years (2000 onward).

## Dataset
The dataset used is from the **US Environmental Protection Agency (EPA)**, containing:
- Year
- CSIRO Adjusted Sea Level
- Lower Error Bound
- Upper Error Bound
- NOAA Adjusted Sea Level

The file is named: `epa-sea-level.csv`.
